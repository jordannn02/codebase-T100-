# Codex Plugin 範本

這個資料夾是公開安全的 plugin scaffold。

使用方式：

1. 先在本機安裝或建置 `codebase-memory-mcp` / `codebase-memory-mcp-pro`。
2. 複製 `.mcp.example.json` 為 `.mcp.json`。
3. 把 `command` 改成你的本機 binary 路徑，或保留 PATH 上的 `codebase-memory-mcp`。
4. 將此資料夾依你的 Codex plugin 安裝方式放到 plugin 目錄。

注意：公開版不附私人 binary、不附正式環境設定、不附 source 索引資料。
